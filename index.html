🧶 맞춤형 뜨개질 도안 생성기

이미지를 픽셀화하여 실제 코 단위의 뜨개질 도안으로 변환해 줍니다.

설정

대바늘은 코가 납작하여 비율 보정이 필요합니다.
가로(cm)
세로 단 수는 선택한 비율과 이미지에 맞춰 자동 계산됩니다.
색
선택한 필수 색상을 포함하여 총 사용할 색상 수입니다.
오른쪽 원본 이미지에서 살리고 싶은 중요한 색상(눈동자, 옷 등)을 클릭하면 고정 색상으로 등록됩니다.
💡 Tip: 버튼을 다시 누르면 추출되는 색상 조합이 미세하게 달라집니다. 마음에 드는 도안이 나올 때까지 여러 번 생성해 보세요!

최근 생성 기록 (클릭하여 복구, 최대 5개)

사용된 색상표 (실 번호)

  • 이미지를 업로드하고 도안을 생성해주세요.
