# Blueprint: SSUESSUE KNITS (쓔쓔닛츠)

**Knitting Pattern Generator & Smart Toolkit for Modern Knitters.**

---

## 🚀 Core Value Proposition
사용자가 업로드한 이미지를 분석하여 게이지와 실 굵기에 최적화된 **나만의 뜨개 도안**을 즉시 생성하고, 단수 카운터와 게이지 계산기를 통해 작업의 전 과정을 스마트하게 보조합니다.

## 🛠️ 핵심 기능 (Key Features)
1.  **AI Pattern Generation**: 
    -   K-means++ 클러스터링 기반 지능형 색상 추출.
    -   Pixel / Filet Crochet 모드 지원 및 PDF 고해상도 출력.
    -   대바늘/코바늘 기법별 비율(Aspect Ratio) 자동 조정.
2.  **Smart Knitting Toolkit**:
    -   사용성 중심의 게이지 계산기 및 균등 증감 알고리즘.
    -   프로젝트별 다중 단수 카운터 (반복 모드 지원).
3.  **Community & Insights**:
    -   철저한 도안 프라이버시 보호 기반의 작품 공유 피드.
    -   3개 국어(KO, EN, JA) 지원 매거진 및 아티클 시스템.

## 🏗️ Technical Architecture
-   **Frontend**: Vanilla JS (Modules), HTML5/CSS3.
-   **Infrastructure**: 
    -   Firebase (Auth, Firestore, Storage) — Serverless BaaS.
    -   Cloudflare Pages — Global Edge Hosting.
-   **SEO/GEO**: `hreflang` 기반 다국어 타겟팅 및 JSON-LD 구조화 데이터 적용.

## 🔗 External Integrations
-   **Analytics**: Google Analytics 4 (G-KSBJ5J8N4K), Microsoft Clarity.
-   **Monetization**: Google AdSense 연동 완료.
-   **Engagement**: Formspree (Customer Inquiries).
