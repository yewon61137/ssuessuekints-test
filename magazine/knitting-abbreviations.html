뜨개 약어 완전 정리
— 코바늘·대바늘·일본 기호 한눈에

2026년 4월 17일  ·  SSUESSUE KNITS

뜨개 도안은 전 세계 공통 약어를 사용하지만, 국가별·언어별로 표기 방식이 조금씩 다릅니다. 특히 북미식 영문 약어와 영국식 약어는 이름이 같은데 뜻이 다른 경우가 있어 혼란을 주기도 합니다. 이 가이드는 코바늘·대바늘 약어를 한국어·영어·일본어로 대조 정리하고, 북미식/영국식 차이와 일본 기호 도안까지 함께 안내합니다. 도안을 펼쳐두고 옆에 놓고 보는 레퍼런스로 활용하세요.

일본 기호 도안 읽기

일본 뜨개 도안은 문자 약어보다 그림 기호를 주로 사용합니다. 주요 기호 의미:

  • □ (빈 사각형): 겉뜨기 (RS에서) / 안뜨기 (WS에서)
  • — (가로 막대): 안뜨기 (RS에서) / 겉뜨기 (WS에서)
  • ○ (동그라미): 걸기코 (yo)
  • \ (왼쪽 대각선): 왼코 줄이기 (ssk)
  • / (오른쪽 대각선): 오른코 줄이기 (k2tog)
  • V (V자): 걸기코 또는 짧은뜨기 (코바늘)
  • T (T자): 한길긴뜨기 (코바늘)
  • X (엑스): 빼뜨기 또는 코 교차

일본 도안에는 항상 범례(凡例, 기호 설명표)가 첨부됩니다. 범례를 먼저 확인하고 작업하는 습관을 들이세요.

도안 구조 약어

  • beg: beginning (시작)
  • rem: remaining (남은 코)
  • approx: approximately (약)
  • inc: increase (늘리기)
  • dec: decrease (줄이기)
  • cont: continue (계속하다)
  • patt: pattern (패턴대로 계속)
  • tog: together (함께, 합쳐서)
  • foll: following (다음의)
  • alt: alternate (번갈아)

Knitting and crochet patterns use standardized abbreviations — but different countries use different conventions. North American (US) and British (UK) crochet abbreviations share the same letters but refer to different stitches, which causes endless confusion. This reference guide covers all major abbreviations for knitting, crochet, and Japanese symbol charts side by side. Bookmark it and keep it open while you work.

Japanese Symbol Charts — Key Symbols

Japanese patterns use visual symbols instead of text abbreviations. Always check the legend (凡例, hanrei) for the specific meanings in each pattern.

  • □ (empty square): knit on RS, purl on WS
  • — (horizontal bar): purl on RS, knit on WS
  • ○ (circle): yarn over (yo)
  • \ (left diagonal): left-leaning decrease (ssk)
  • / (right diagonal): right-leaning decrease (k2tog)
  • V (crochet): single crochet or yarn over
  • T (crochet): double crochet / 長編み

  • alt: alternate  ·  rep: repeat  ·  sts: stitches

編み物の編み図は世界共通の略語を使いますが、国や言語によって表記が異なります。特に北米式（US）と英国式（UK）のかぎ針略語は、同じ文字が違う技法を指すことがあり混乱しやすいです。このガイドでは、かぎ針・棒針の略語を日本語・英語・韓国語で対照整理し、北米式/英国式の違いも解説します。編み図を開きながら手元に置くリファレンスとして活用してください。

💡 US式とUK式の最大の罠！ US式「sc」（細編み）＝ UK式「dc」、US式「dc」（長編み）＝ UK式「tr」。同じ文字で全く違う編み目を指すため、パターンの表記方式を必ず確認してから編み始めましょう。

日本の記号図の主な記号

  • □（空白マス）：表目（表面）/ 裏目（裏面）
  • —（横棒）：裏目（表面）/ 表目（裏面）
  • ○（丸）：掛け目（yo）
  • \（左斜め）：左上2目一度（ssk）
  • /（右斜め）：右上2目一度（k2tog）
  • V：細編みまたは掛け目（かぎ針）
  • T：長編み（かぎ針）

日本の編み図には必ず凡例（はんれい）が付いています。作業前に必ず凡例を確認しましょう。
