# 🧶 쓔쓔닛츠 (SSUESSUE KNITS)

**뜨개인을 위한 스마트 올인원 플랫폼 — Smart All-in-One Platform for Knitters**

SSUESSUE KNITS는 전통적인 뜨개질 방식에 현대적인 기술을 결합하여, 누구나 사진 한 장으로 자신만의 도안을 만들고 관리할 수 있는 창작 환경을 제공합니다.

---

## ✨ 주요 기능 (Key Features)

### 🖼️ AI 도안 생성기 (AI Pattern Generator)
*   **이미지 → 도안 변환**: 업로드한 사진을 k-means 알고리즘으로 분석하여 실 굵기와 게이지에 최적화된 컬러 도안을 자동 생성합니다.
*   **기법 최적화**: 코바늘(1:1) 및 대바늘(5:7/7:5) 비율을 지원하며, 픽셀 아트 및 방안뜨기(Filet Crochet) 모드를 제공합니다.
*   **PDF/PNG 내보내기**: 생성된 도안을 고해상도 문서로 저장하여 오프라인에서도 확인할 수 있습니다.

### 📐 스마트 뜨개 도구함 (Smart Knitting Toolkit)
*   **게이지 계산기**: 스와치 데이터를 기반으로 목표 크기에 필요한 코/단 수를 즉시 산출합니다.
*   **단수 카운터 (Row Counter)**: 여러 프로젝트를 동시에 관리하며, 반복 패턴을 위한 전용 모드를 지원합니다.
*   **균등 늘림/줄임 계산**: 복잡한 공식 없이 몇 번의 클릭만으로 균등한 코 조절 지침을 제공합니다.

### 🏠 커뮤니티 & 매거진 (Community & Magazine)
*   **작품 공유**: 전 세계 뜨개인들과 자신의 작품을 공유하고 소통합니다. (도안 프라이버시 보호 기능 포함)
*   **매거진**: 뜨개질의 역사, 과학, 트렌드를 다루는 유익한 아티클을 3개 국어로 제공합니다.

---

## 🛠️ 기술 스택 (Tech Stack)

*   **Core**: Vanilla JavaScript (ES Modules), HTML5, CSS3
*   **Backend / BaaS**: Firebase (Authentication, Firestore, Cloud Storage)
*   **Infrastructure**: Cloudflare Pages (Deployment), Formspree (Contact)
*   **Libraries**: jsPDF (PDF Generation)
